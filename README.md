## 🔧 About This Repository

This is the **active development repository** for ELPMS. All new features, 
bug fixes, and experimental changes are developed and tested here before 
being merged into the main repository.

## 📌 Development Workflow

This repository exists to:

- Isolate active development from the stable production code
- Allow features and fixes to be tested before release
- Keep the main repository clean and deployment-ready

Changes are only pushed to the main repository once:

- ✅ Development is complete
- ✅ Features are fully tested
- ✅ Fixes are verified and stable
